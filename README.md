# Mall Customers Clustering (Unsupervised Machine Learning)
## Project Description
The main goal of this project is to practice and enhance the skill and knowledge on clustering techniques (unsupervised machine learning). This dataset contains the data of customers from a supermarket mall like Customer ID, age, gender, annual income and spending score (assigned based on defined parameters like customer behavior and purchasing data). I'll analyze, clean, preprocess and scale to finally build two clustering models, KMeans Clustering and Agglomerative Clustering, and compare the performance. Finally, the results could be use to group customers and offer personalized offers to each one based on the annual income and spending score of the segmentation.
## Goal
Practice the unsupervised machine learning algorithms, clustering techniques.
## Content
* Exploratory Data Analysis (EDA)
* Scaling features and Elbow method
* Clustering models
* Results and conclusions
